Lista de Exercícios - HTML

Estes exercícios ajudarão você a entender e praticar os fundamentos do HTML.

Exercícios:

  1. Crie a estrutura básica de um documento HTML5.
  2. Insira um título com a tag <h1>.
  3. Crie uma lista não ordenada com 3 itens.
  4. Adicione uma imagem com a tag <img>.
  5. Crie um link para outro site usando a tag <a>.
  6. Monte um formulário simples com nome e email.
  7. Adicione um rodapé com a tag <footer>.
